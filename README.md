# pwa-cabin-of-suicides

## Installation front
```
cd client && npm i
```

## Installation back
```
cd server && npm i
```

Copier le `.env.dist` en `.env`.

Pour certains paramètres, me contacter.

## Lancement dev front
```
cd client && npm start
```

## Lancement dev back
```
cd server && npm run start:dev

# si erreur sur le port 5000 car déjà en écoute
cd server && npm run start:dev:kill
```
